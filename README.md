# hexo-blog

[mrzjd.cn](http://mrzjd.cn) 博客仓库

branch master - article code
branch web - dist
branch theme - theme code

#### Dev Env

* hexo-blog [master] - 用于写文章
* hexo-blog-web [web] - website
* hexo-blog-imgs

```bash
cd hexo-blog

hexo server

hexo generate

. publish.bat # 将生成的html移入website下
```
